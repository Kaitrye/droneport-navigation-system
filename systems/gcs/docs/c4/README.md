# C4 диаграммы НУС

## Состав

- `C1_Context.puml` — контекст системы (внешние акторы и границы НУС).
- `C2_Containers.puml` — контейнерный уровень НУС.
- `C3_RobotSystem_Components.puml` — компонентный уровень RobotSystem.
- `S1_DroneService_Interaction.puml` — последовательность взаимодействия НУС с внешним сервисом дронов (команды/ответы/телеметрия).
- `Drones.md` — текстовый контракт интеграции с внешним сервисом дронов (команды, ответы, события, ошибки, NFR).
- `S2_DronePort_Interaction.puml` — последовательность взаимодействия НУС с Дронопортом (подготовка, выдача, приём, аварийный сценарий).
- `DronePort.md` — текстовый контракт интеграции с Дронопортом (команды, ответы, ошибки, NFR).

## Как открыть

1. Установите VS Code расширение PlantUML.
2. Откройте нужный `.puml` файл.
3. Вызовите команду `PlantUML: Preview Current Diagram`.

## Примечание

Диаграммы используют библиотеку C4-PlantUML по URL include. Если в сети есть ограничения, подключите библиотеку локально и замените `!include` на локальный путь.
